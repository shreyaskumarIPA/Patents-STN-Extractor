About this tool

The tool:

  • extracts patent publication numbers from supported STN fields and formats jurisdiction specific output suitable to be copied and pasted to EPO AbS.
  • extracts PubMed identifiers for Medline and Embase output blocks.
  • extracts WPIDS accession numbers and formats them for STN search syntax.
  • keeps the same extraction rules and automatic updating behaviour as the original tool.

Copy and paste the results:

  • directly into the intended tool.
  • results are de-duplicated and sorted for readability.
  • where each output panel includes a compact one-click copy to clipboard control.
